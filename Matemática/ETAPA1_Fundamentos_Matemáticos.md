# Operações com números reais
___
## Resumo sobre números reais.
O conjunto dos números racionais (aqueles que podem ser represrentados em fração), com os irracionais, (aqueles que não podem ser representados em fração). 

Podemos calcular 4 principais operações com números reais, sendo elas adção, subtração, dvisão e multiplicação. Além de potenciação e radiciação, que são conjuntos complementares.
___
## Os números reais possuem uma série de propriedades:

### Elemento neutro:  
Na adição: O número 0: n+0=n  
Na multplicação: O número 1: n*1=n  

### Propriedade comutativa:  
A ordem não altera o resultado  
Ex: n+m = m+n // n.m = m.n  

### Propriedade associativa:  
A forma de agupar não altera o resultado  
Ex: (a+m)+n = (n+m)+a // a.(m.n) = m.(n.a)  

### Propriedade distrbutiva:  
O produto da soma é igual a soma dos produtos:  
Ex: a.(m+n) = a.m + a.n  

### Existência do inverso multiplicador:  
Para todo número real diferente de zero (n!=0) existe o inverso multiplicador. Tal que n.(1/n) = 1  

Ex:  
Inverso de 5 = 1/5  
inverso de -3 = -(1/3)  
inverso de 2/7 = 7/2  
inverso de $\sqrt{2}$ = 1 / $\sqrt{2}$  

### Fechamento:
A soma ou multiplicação de dois números reais resulta sempre em um número real.

## Números naturais:
O primeiro conjunto nuérico a surgir, atende a necessidade dos seres humanos de contabilizar e quantificar objetos.  
Ex: N = 0,1,2,3,4,5...  

## Números inteiros:  
Inclui os números negativos.  
Ex: -3,-2,-1,0,1,2,3...  

## Números racionais:  
É formado por todos os números que podem ser escritos nafrma de fração, nele incluímos os números decimais, dízimas periódicas e números inteiros. Diferentemente dos conjuntos supracitados, o conjunto dos números racionais não pode ser ordenado em uma lista crescente na reta numérica, pois entre dois números racionais distintos sempre existe outro número racional.    
Ex: entre 0 e 1, temos 0,5; entre 0 e 0,5, temos o 0,25; entre 0 e 0,25, temos 0,125, e assim sucessivamente.   
Os números racionais são representados pela letra **Q**.

## Números irracinais
É formado pelos números que não podem ser escritos na forma de fração. São rízes não exatas e dizimas não periódicas.  

Ex: $\sqrt{3}$ (Não exata)  
π = 3,14159… (O π é uma dízima não periódica, logo, é um número irracional.)  
7,12344219023... (Dízima não periódica)

## Representação dos números reais na reta numérica:  
Podemos representar os números reais em uma reta, isso possibilita, por exemplo, o desenvolvimento da área da geometria analítica e o estudo de gráficos de funções.

Para essa representação, utilizamos o princípio da boa ordem dos números reais, pois, ao comparar dois números reais distintos, é possível saber qual deles é o maior e então ordená-los. A representação dos números na reta é conhecida como reta numérica ou reta real. O marco zero é conhecido como origem da reta. Para o lado direito, temos os números reais positivos, e, para o lado esquerdo, os números reais negativos.
