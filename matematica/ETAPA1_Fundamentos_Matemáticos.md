# Operações com números reais
___
## Resumo sobre números reais
O conjunto dos números reais é formado pelos números racionais (aqueles que podem ser representados em forma de fração) e pelos números irracionais (aqueles que não podem ser representados em forma de fração).

Podemos realizar quatro operações principais com números reais: **adição, subtração, divisão e multiplicação**. Além dessas, temos a **potenciação** e a **radiciação**, que são operações complementares.
___
## Os números reais possuem uma série de propriedades:

### Elemento neutro  
Na adição: o número **0**  
n + 0 = n  

Na multiplicação: o número **1**  
n · 1 = n  

### Propriedade comutativa  
A ordem dos fatores não altera o resultado.  
Ex:  
n + m = m + n  
n · m = m · n  

### Propriedade associativa  
A forma de agrupar não altera o resultado.  
Ex:  
(a + m) + n = a + (m + n)  
a · (m · n) = (a · m) · n  

### Propriedade distributiva  
O produto da soma é igual à soma dos produtos.  
Ex:  
a · (m + n) = a · m + a · n  

### Existência do inverso multiplicativo  
Para todo número real diferente de zero (n ≠ 0), existe um inverso multiplicativo, tal que:  
n · (1/n) = 1  

Ex:  
Inverso de 5 = 1/5  
Inverso de −3 = −1/3  
Inverso de 2/7 = 7/2  
Inverso de √2 = 1/√2  

### Propriedade do fechamento  
A soma ou a multiplicação de dois números reais resulta sempre em um número real.  
___
## Números naturais  
É o primeiro conjunto numérico a surgir, atendendo à necessidade dos seres humanos de contabilizar e quantificar objetos.  
Ex:  
N = {0, 1, 2, 3, 4, 5, …}  

## Números inteiros  
Inclui os números negativos, além do zero e dos positivos.  
Ex:  
Z = {…, −3, −2, −1, 0, 1, 2, 3, …}  

## Números racionais  
É formado por todos os números que podem ser escritos na forma de fração. Nesse conjunto estão incluídos os números decimais, as dízimas periódicas (como 3,141414…, em que os números se repetem) e os números inteiros.

Diferentemente dos conjuntos supracitados, o conjunto dos números racionais não pode ser descrito por uma lista ordenada, pois entre dois números racionais distintos sempre existe outro número racional.  

Ex:  
Entre 0 e 1, temos 0,5;  
Entre 0 e 0,5, temos 0,25;  
Entre 0 e 0,25, temos 0,125;  
E assim sucessivamente.  

Os números racionais são representados pela letra **Q**.

## Números irracionais  
É formado pelos números que não podem ser escritos na forma de fração. São raízes não exatas e dízimas não periódicas.  

Ex:  
√3 (raiz não exata)  
π = 3,14159… (π é uma dízima não periódica, logo, é um número irracional)  
7,12344219023… (dízima não periódica, em que os números não se repetem.)  

## Representação dos números reais na reta numérica  
Os números reais podem ser representados em uma reta, o que possibilita, por exemplo, o desenvolvimento da geometria analítica e o estudo de gráficos de funções.

Para essa representação, utilizamos o princípio da boa ordem dos números reais, pois, ao comparar dois números reais distintos, é possível determinar qual deles é o maior e, assim, ordená-los. Essa representação é conhecida como **reta numérica** ou **reta real**.  

O ponto zero é chamado de **origem da reta**. À direita da origem estão os números reais positivos e, à esquerda, os números reais negativos.
